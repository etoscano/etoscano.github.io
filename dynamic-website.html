Description of the project
In this project I've created a dynamic website meant for a voluntary association. IDM submodels have been used for Content, Navigation and Presentation design and interaction scenarios have been created to describe the flow of usage.
The images above display the frontal and the floor projections of the three games inside GEA++.

The interaction for the first two games is based on hand tracking while the third game relies on the pyramid smart object.

The image on the right shows the pyramid smart object that we have created for the third game.
